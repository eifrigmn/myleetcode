# [3的幂](https://leetcode-cn.com/leetbook/read/top-interview-questions-easy/xnsdi2/)

## 题目

给定一个整数，写一个函数来判断它是否是 `3` 的幂次方。如果是，返回 `true` ；否则，返回 `false` 。

整数 `n` 是 `3` 的幂次方需满足：存在整数 `x` 使得 `n == 3^x`

### 示例

+ 示例1：

~~~
输入：n = 27
输出：true
~~~

+ 示例2：

~~~
输入：n = 0
输出：false
~~~

+ 示例3：

~~~
输入：n = 9
输出：true
~~~

+ 示例4：

~~~
输入：n = 45
输出：false
~~~

### 提示

+ -2^31 <= n <= 2^31 - 1

### 进阶

+ 你能不使用循环或者递归来完成本题吗？
