# [最长公共前缀](https://leetcode-cn.com/explore/interview/card/top-interview-questions-easy/5/strings/40/)

## 题目

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串`""`。

### 示例

+ 示例1：

```
输入: ["flower","flow","flight"]
输出: "fl"
```

+ 示例2：

```
输入: ["dog","racecar","car"]
输出: ""
解释: 输入不存在公共前缀。
```

### 说明

所有输入只包含小写字母`a-z`。
