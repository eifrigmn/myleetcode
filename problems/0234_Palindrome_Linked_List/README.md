## [回文链表](<https://leetcode-cn.com/problems/palindrome-linked-list/>)

### 题目

请判断一个链表是否为回文链表。

**示例1：**

```
输入: 1->2
输出: false
```

**示例2：**

```
输入: 1->2->2->1
输出: true
```

**进阶：**

你能否用 O(n) 时间复杂度和 O(1) 空间复杂度解决此题？

### 思路

+ 配合数组

  把链表中的值放到数组中

  分头从两边遍历数组

  直到数组中位，没有出现不等的数据，则为回文链表，反之则不是回文链表

+ 将原始链表翻转，然后顺序遍历两个链表

+ 使用快慢指针法找到链表中点，然后逐个比较
